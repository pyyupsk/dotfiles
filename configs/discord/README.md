# Discord Configuration

This directory provides custom CSS for enhancing your Discord experience using the Everblush theme. You can apply these customizations through Vencord, BetterDiscord, or Stylus.

## Applying Custom CSS with Vencord

### Installation and Setup

1. **Install Vencord**: Follow the installation instructions on the [Vencord FAQ](https://vencord.dev/faq/#How-do-I-install-Vencord?) page.
2. **Open User Settings**: Launch Discord and go to User Settings.
3. **Access Themes**: Navigate to the "Themes" section under the "Vencord" category in the sidebar.

### Importing Custom CSS

You have three options for importing the custom CSS:

#### 1. Local Themes

**Method 1: Using the Themes Folder**

1. Click on the "Open Themes Folder" button.
2. Place the `everblush.theme.css` file into this folder. You can download the file from the [`configs/discord`](https://github.com/pyyupsk/dotfiles/tree/main/configs/discord) directory.

**Method 2: Using Quick Edit CSS**

1. Open the [`everblush.theme.css`](https://github.com/pyyupsk/dotfiles/blob/main/configs/discord/everblush.theme.css) file in your browser.
2. Copy the entire contents of the file.
3. Paste the copied CSS into the Quick Edit CSS box in Vencord.

#### 2. Online Themes

**Method 3: Using Online Themes**

1. Copy the raw URL of the `everblush.theme.css` file:
    - URL: `https://raw.githubusercontent.com/pyyupsk/dotfiles/main/configs/discord/everblush.theme.css`
2. Paste this URL into the "Online Themes" box in Vencord.

## Applying Custom CSS with BetterDiscord

### Installation and Setup

1. **Install BetterDiscord**: Follow the [BetterDiscord Installation Guide](https://docs.betterdiscord.app/users/getting-started/installation) for setup instructions.
2. **Download the Theme File**: Get the `everblush.theme.css` file from the GitHub repository.
    - URL: [`everblush.theme.css`](https://github.com/pyyupsk/dotfiles/blob/main/configs/discord/everblush.theme.css)
3. **Move the Theme File**: Transfer the downloaded file to the BetterDiscord themes folder:
    - Path: `%appdata%\betterdiscord\themes`

## Applying Custom CSS with Stylus

### Installation and Setup

1. **Install Stylus**: Add the Stylus browser extension from [GitHub](https://github.com/openstyles/stylus?tab=readme-ov-file#releases).
2. **Access Userstyles**: Visit the [Everblush Stylus styles for Discord](https://github.com/pyyupsk/Everblush/tree/main/userstyles/styles/discord) repository.
3. **Install the Style**:
    - Click on the Stylus Install badge to add the Everblush Discord style to Stylus.

## Enabling the Theme

After applying the theme using one of the methods above:

-   For **Online Themes** or **Quick Edit CSS**, the theme should activate automatically.
-   For **Local Themes** (via Vencord’s or BetterDiscord’s themes folder), you may need to manually enable the theme in the themes section of the respective application.

Enjoy your enhanced Discord experience with the Everblush theme!
