# Terminal Configuration

This directory contains the configuration files for customizing your terminal experience, including color schemes and font settings.

## Color Scheme

The color scheme used is **Everblush**, a dark, vibrant, and beautiful color scheme. Below is the JSON configuration for the Everblush color scheme:

```json
{
    "name": "Everblush",
    "cursorColor": "#dadada",
    "selectionBackground": "#31363a",
    "foreground": "#dadada",
    "background": "#141b1e",
    "black": "#232a2d",
    "red": "#e57474",
    "green": "#8ccf7e",
    "yellow": "#e5c76b",
    "blue": "#67b0e8",
    "purple": "#c47fd5",
    "cyan": "#6cbfbf",
    "white": "#b3b9b8",
    "brightBlack": "#5c6268", // I edited this color to increase the contrast.
    "brightRed": "#ef7e7e",
    "brightGreen": "#96d988",
    "brightYellow": "#f4d67a",
    "brightBlue": "#71baf2",
    "brightPurple": "#ce89df",
    "brightCyan": "#67cbe7",
    "brightWhite": "#bdc3c2"
}
```

Thank [Everblush](https://github.com/Everblush/terminal-emulators) for creating this stunning color scheme 💖.

## Nerd Fonts

To enhance your terminal with awesome icons and better font support, we use Nerd Fonts. You can download the latest versions from the [Nerd Fonts Downloads](https://www.nerdfonts.com/font-downloads) page.

## Usage

### Color Scheme

To apply the Everblush color scheme:

1. Open your terminal settings.
2. Navigate to the color scheme or appearance section.
3. Copy and paste the JSON configuration above into the appropriate section for custom color schemes.
4. Save your settings.

### Nerd Fonts

1. Download your preferred Nerd Font from the [Nerd Fonts Downloads](https://www.nerdfonts.com/font-downloads) page.
2. Install the font on your system.
3. Open your terminal settings.
4. Navigate to the font or text section.
5. Select the installed Nerd Font.
6. Save your settings.

## Configuration

For further customization, you can modify the following settings:

-   **Color Scheme**: Adjust the colors in the JSON configuration to suit your preferences.
-   **Fonts**: Experiment with different Nerd Fonts to find the one that best fits your style and needs.
